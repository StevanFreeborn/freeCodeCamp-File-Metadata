File Metadata Microservice

Documentation

Welcome to the File Metadata service. This documentation should supply you with all the information you need to start making requests. It is a really simple api service built to complete one of the projects for the freeCodeCamp Back End Development and APIs certification.

Base Url
https://freecodecampfilemetadata.herokuapp.com
File Metadata Endpoint

An endpoint that will accept a file and return a JSON object with some of the files metadata.

/api/fileanalyse
Example Request:
POST https://freecodecampfilemetadata.herokuapp.com/api/fileanalyse

Content-Type:
multipart/form-data

upfile:
The file to be uploaded.
Example Response:
{
 "name": "Test Attachment.txt",
 "type": "text/plain",
 "size": 14,
}
